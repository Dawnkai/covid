# Atom Infection Simulator

### Language : Python

### GUI : Pygame & Tkinter

Basic simulator for atom collision - based on CoVid pandemic.
Graphically presented using [Pygame](https://www.pygame.org/news) and [Tkinter](https://docs.python.org/3/library/tkinter.html).
