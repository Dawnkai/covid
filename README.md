# Atom Infection Simulator

### Language : Python

### GUI : Tkinter & Pygame

Basic simulator for atom collision - based on CoVid pandemic.
Graphically presented using [Pygame](https://www.pygame.org) and [Tkinter](https://docs.python.org/3/library/tkinter.html).
